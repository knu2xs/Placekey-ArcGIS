placekey_arcgis.get_placekey_from_address(street_address, city, state, postal_code, iso_country_code='US', placekey_api_key=None)¶

Look up the full Placekey for a given address string.

Parameters

  • street_address (str) – Street address with suite, floor, or apartment.

  • city (str) – The city.

  • state (str) – Two character state identifier.

  • postal_code (str) – Postal code identifier; typically five numbers.

  • iso_country_code (str) – Two character country identifier. Defaults to “US”.

  • placekey_api_key (Optional[str]) – Placekey API key for making requests.

Return type

str

Returns

Placekey string.

placekey_arcgis.get_placekey_from_geometry(geometry, placekey_api_key=None)¶

Lookup the full Placekey for a given Esri ArcGIS Python API Geometry object. If a point, obivously the coordinates for the point will be used. However, if a line or a polygon, the coordinates at the centroid will be used.

Parameters

  • geometry (Geometry) – ArcGIS Python API geometry.

  • placekey_api_key (Optional[str]) – Placekey API key for making requests.

Return type

str

Returns

String with full Placekey for given location.
